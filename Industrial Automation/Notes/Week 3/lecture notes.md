# Valves

valces act as control components just as a relay or a transistor can exercise control in a circuit. They require external actuation but they exercise that logic.

### Logical valces

shuttle valve

| Name          | Logic | Symbol |
| ------------- | ----- | ------ |
| Check valve   | MAX   |        |
| Double Check  | OR    |        |
| shuttle valve | XOR   |        |
| Dual Pressure | AND   |        |
